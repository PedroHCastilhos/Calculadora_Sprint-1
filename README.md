Grupo de Estudos da AGES com a Preto no Branco

Sprint 1

Calculadora

  Essa primeira sprint foi pensada para desenvolver habilidades iniciais em HTML, CSS e Javascript através do desenvolvimento de uma Calculadora funcional. O HTML é utilizado para indicar os elementos que estão contidos na calculadora, o CSS desenvolve a parte do visual da calculadora e o Javascript tem o papel de desenvolvimento da parte do backend da calculadora.
